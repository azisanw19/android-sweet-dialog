How to Use

### Step 1. Add the JitPack repository to your build file

```groovy
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

### Step 2. Add the dependency

```groovy
	dependencies {
	        implementation 'com.github.azisanw19:android-sweet-dialog:Tag'
	}
```
